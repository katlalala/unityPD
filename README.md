# Ģērbšānās spēlīte (Unity 2D)
Interaktīvs 2D saģērbšanas projekts kas izstrādāts Unity vidē. Spēlētājs var pielāgot tēlus, izvēlēties apģērbu, mainīt tēla izmērus un ievadīt informāciju par tēlu.

## Galvenās funkcijas
- **Tēlu pārslēgšana**: izmantojot Dropdown var pārslēgties starp 2 tēliem
- **Drag and drop sistēma**: Apģērbu un aksesuārus var pārvietot ar pelītes palīdzību un uzlikt uz tēla ar snap to position.
- **Tēla informācija un vecuma aprēķins**
- **Skaņas efekti**: Uzklikšķinot uz tēla, tiek atskaņots audio klips. Abiem tēliem ir atšķirīgi audio.
- **Fade in un out starp scenes**
